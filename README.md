# EMUS Website (TanStack Migration)

This project is a React + TanStack migration of the previous EMUS static site with route and visual parity.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Routes

- `/`
- `/team`
- `/project`
- `/sponsors`
- `/gallery`
- `/contact`
- `/abatiello`

## Notes

- Assets live under `public/assets/`.
- Route files live under `src/routes/`.
- Shared site data (team/contact) lives in `src/data/site.ts`.
